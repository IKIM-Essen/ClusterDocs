# Imported RCC user-workflow material

The v0.1.2 course expansion uses the Python, R, Jupyter and Shiny workflow material supplied with the latest RCC tarball.

The uploaded archive did not contain a literal top-level directory named `move_to_user_docs` after extraction in the validation environment. The relevant user-facing material was present under:

- `docs/user-guides/python-r-shiny-jupyter.md`
- `examples/user-workflows/python/`
- `examples/user-workflows/r/`
- `examples/user-workflows/jupyter/`
- `examples/user-workflows/shiny/`

That material was adapted into ClusterDocs rather than copied verbatim into public pages. Internal operational configuration, infrastructure topology, and administrator-only details were not published.
